# Portfólio 3D — Cidade Analítica

Projeto React + Vite + Three.js com React Three Fiber.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build para Hostinger ou GitHub Pages

```bash
npm run build
```

Publicar a pasta `dist`.

## Estrutura
- `src/App.jsx`: layout e painel lateral
- `src/components/CityScene.jsx`: cena 3D principal
- `src/cityData.js`: conteúdo dos distritos
